How to use:
1. Make sure you install node.js.
2. clone the tweet-saver repository, run "npm install"  to grab the dependencies.


Running the app:
Runs like a typical express app: "node server.js"

How to play:
1.open your browser, and use "localhost:3030" as the url in the browser.
2. enter any search keyword in the search bar, and relevant tweets stream will come out (10 at once)
